Overview

This program was written in my spare time to brush up on my C#. I figured a maze was complex enough to be an interesting design challenge without going overboard.

Design

The basic idea was to create libraries that provided set pieces. Other libraries would build off of the original set pieces or provide additional functionality. Only the top level project would give the program any real form.

Features

Mazes
The cornerstone of Beautiful Mazes. The mazes hold the layout and topology of the maze.
Characters
The entities that live in the maze. They control their own movement patterns and restrictions. This allows multiple character types to live in the maze. Want a ghost that can pass through walls? No problem.
Random Generators
Several (two) kinds of random maze generators create unique mazes.
Renderers
Various Renderers can be utilized to visualize the maze how you see fit. Currently, the only renderers available are text renderers with customizable character sets. A special kind of camera renderer allows the maze to be cropped around a certain point.

Screenshots

This is a top level program designed to showcase all of the features available so far. There are two kinds of mazes: wall maze and block mazes. Use WASD to move the character around. There is no "beginning" or "end" as events have not been created yet.
